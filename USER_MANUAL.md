### 4. 🔬 因子分析

**功能**: 分析 Alpha158 因子的有效性（IC 分析）

#### 参数设置

| 参数 | 说明 | 建议值 |
|------|------|--------|
| 开始日期 | 分析起点 | 最近1年 |
| 结束日期 | 分析终点 | 数据最新日期-10天 |
| 预测周期 | IC 计算周期 | 5天 |

#### 输出内容

1. **因子 IC 图**: Rank IC 排名
   - 正值 (绿色): 因子值高 → 未来收益高 (买入信号)
   - 负值 (红色): 因子值高 → 未来收益低 (卖出信号)

2. **因子分布图**: 显示因子的截面分布

**核心概念 - IC (Information Coefficient)**:

| IC 值 | 含义 | 结论 |
|-------|------|------|
| > 0.05 | 强因子 | 预测能力强 |
| 0.02-0.05 | 有效因子 | 有一定预测能力 |
| < 0.02 | 弱因子 | 预测能力弱 |

**使用场景**:
- 了解哪些因子当前有效
- 验证投资逻辑
- 指导因子选择

### 5. ⚡ 模型回测

**功能**: LightGBM + Alpha158 因子回测，生成操作建议

#### 数据设置

| 参数 | 默认值 | 说明 |
|------|--------|------|
| 训练集开始 | 2008-01-01 | 模型训练起始日 |
| 训练集结束 | 2016-12-31 | 模型训练结束日 |
| 测试集开始 | 2019-01-01 | 回测验证起始日 |
| 测试集结束 | 数据最新-10天 | 回测验证结束日 |

#### 策略参数 (个人投资者模式)

| 参数 | 默认值 | 说明 |
|------|--------|------|
| 持仓股数 | 7只 | 同时持有的股票数量 |
| 每次换手 | 1只 | 每次调仓换掉的股票数 |

#### 回测指标

| 指标 | 说明 | 目标值 |
|------|------|--------|
| Sharpe | 夏普比率 (风险调整后收益) | > 1 |
| 年化收益 | 策略年化收益率 | 越高越好 |
| 最大回撤 | 最大亏损幅度 | < 20% |
| 胜率 | 盈利交易日占比 | > 50% |
| Calmar | 年化收益/最大回撤 | > 2 |

#### 输出内容

1. **回测指标**: 6 个核心指标
2. **净值曲线**: 策略 vs 基准对比
3. **回撤曲线**: 资金曲线回撤
4. **本周操作建议**:
   - 建议买入 (Top K)
   - 建议卖出/避开 (Bottom K)
   - 预测得分和预期收益

**使用场景**:
- 验证策略有效性
- 获取具体买卖建议
- 评估风险收益

## ❓ 常见问题

### Q1: 数据更新频率？

**A**: 建议每周更新一次。点击 **🔥 主题热点** 页面的 **🔄 更新 Qlib 数据** 按钮。

### Q2: 回测结果和实际交易差异？

**A**: 
- 回测假设: 每天调仓、瞬间成交、无滑点
- 实际交易: 每周调仓、有滑点、有时间差

**建议**: 按回测结果的 50-70% 预期实际收益。

### Q3: 为什么有些股票没有因子评分？

**A**: 只显示 CSI300 成分股的因子评分。其他股票需要扩展数据源。

### Q4: 数据存储在哪里？

**A**: `~/.qlib/qlib_data/` 目录下，包括:
- `cn_data/` - CSI300 历史数据
- `cn_data_akshare/` - ETF 数据

### Q5: 可以添加自己的股票吗？

**A**: 可以编辑 `stock_universe.py` 文件，添加自定义股票代码。

### Q6: 服务启动失败怎么办？

**A**: 
```bash
# 检查端口占用
lsof -i :8501

# 停止旧进程
pkill -f streamlit

# 重新启动
cd ~/projects/qlib-workspace
./start.sh
```

### Q7: 预测准确率如何？

**A**: 
- Sharpe > 1: 策略有效
- Sharpe 0.5-1: 策略一般
- Sharpe < 0.5: 策略需要优化

实际收益约为回测收益的 50-70%。

### Q8: 可以用于实盘交易吗？

**A**: 
- ✅ 可以作为决策参考
- ❌ 不建议完全自动化交易
- ⚠️ 交易前请自行验证和风控

## 📌 关键概念

### Alpha158 因子

Qlib 提供的 158 个技术因子，包括:
- 价格动量
- 波动率
- 成交量变化
- 相对强弱
- ...

### IC (Information Coefficient)

因子与未来收益的相关系数:
- IC > 0: 正相关 (因子值高 → 未来收益高)
- IC < 0: 负相关 (因子值高 → 未来收益低)
- |IC| > 0.05: 强因子

### Sharpe 比率

风险调整后的收益:
- Sharpe = 年化收益 / 年化波动率
- Sharpe > 1: 优秀的策略
- Sharpe > 2: 卓越的策略

### 最大回撤

从高点到低点的最大亏损:
- 回撤 < 10%: 低风险
- 回撤 10-20%: 中等风险
- 回撤 > 20%: 高风险
